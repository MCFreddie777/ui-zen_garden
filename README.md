## Inštalácia

Prerekvizity:

-   [Node.js](https://nodejs.org)
-   [Node package manager](https://npmjs.com)

V root adresári spustite nasledovné príkazy:

```
npm install
```

A následne, pre každú transpiláciu typescriptového kódu na javascript a spustenie kódu v node.js:

```
npm run dev
```
